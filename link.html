Ini adalah halaman links.

Cara membuat link untuk pindah ke halaman lain yaitu dengan menggunakan tag <a> diikuti atribut href didalam tag <a> untuk menentukan lokasi halaman web yang dituju, contohnya sebagai berikut.

<a href="https://www.google.com/">Link</a>

Hasilnya adalah sebagai berikut, link dibawah jika di klik akan memindahkan kita ke halaman situs google.
