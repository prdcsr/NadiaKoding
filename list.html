Ini adalah halaman lists.

Ada beberapa jenis list dalam HTML, diantaranya adalah ordered list dan unordered list. Untuk membuat ordered list yaitu dengan tag <ol> dan untuk membuat unordered list yaitu dengan tag <ul>. Kemudian di dalam tag tersebut kita harus menambahkan tag <li> untuk menampilkan tiap daftar, contohnya sebagai berikut.

Contoh Ordered List

<ol>
<li>Kopi</li>
<li>Teh</li>
<li>Susu</li>
</ol>

Hasilnya adalah sebagai berikut.

  1. Kopi
  2. Teh
  3. Susu

Contoh Unordered List

<ul>
<li>Kopi</li>
<li>Teh</li>
<li>Susu</li>
</ul>

Hasilnya adalah sebagai berikut.

  • Kopi
  • Teh
  • Susu
